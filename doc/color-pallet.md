# Color Palette

## Primary Colors

### Green
- **Hex**: #84994F
- **Usage**: Primary brand color, main accents, primary buttons

### Yellow
- **Hex**: #FFE797
- **Usage**: Secondary accents, highlights, warning states

### Orange
- **Hex**: #FCB53B
- **Usage**: Call-to-action elements, important highlights

### Red/Brown
- **Hex**: #A72703
- **Usage**: Error states, danger elements, strong accents

## Color Usage Guidelines

### Primary Branding
- Use #84994F for primary buttons and main navigation elements
- Use #FFE797 for secondary buttons and subtle highlights

### Interactive Elements
- Use #FCB53B for call-to-action buttons and important interactive elements
- Use #A72703 sparingly for error states and critical alerts

### Backgrounds and Text
- Primary text: Dark colors (#A72703 or similar dark neutrals)
- Secondary text: Medium gray tones
- Backgrounds: Light neutrals or white
- Accent backgrounds: Use #FFE797 for subtle highlights

### Accessibility
- Ensure sufficient contrast between text and background colors
- Test color combinations for WCAG compliance
- Consider colorblind-friendly alternatives where needed

## Color Combinations

### Safe Combinations
- #84994F on white background
- #A72703 on light backgrounds
- #FCB53B on dark backgrounds
- #FFE797 as background with dark text

### Avoid
- #FFE797 text on #FCB53B background (low contrast)
- #84994F text on #A72703 background (low contrast)
